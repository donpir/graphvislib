The aim of this page is to present three visualisations to represent plagiarism among songs. Our aim is to search for an intuitive visualisation to easily identify plagiarism with a dataset of artifacts (e.g., songs, text, etc.). The following are four visualisations, which use specially prepared dataset of songs. They have been test with end-users.

Note: This page is compatible with the following browsers: Chrome (version >= 49), Firefox (version >= 44), Safari and Opera (version >= 36).

Demo of the visualizations

  • Demo Viz1 - It uses red markers on the edge to represent common notes.
  • Demo Viz2 - Similarity proportional to the edges' thickness. Other variables fixed.
  • Demo Viz3 - Similarity inversely proportional to the distance of vertex. Other variables fixed.
  • Demo Viz4 - Similarity represented as dashed edges

Evaluation study

The three visualisations have been evaluated with 33 participants who performed tasks in random order. The questionnaire is available here. The tasks performed are available here in paper format. These tasks during the evaluation phase have been presented in random order using the open source framework available VizTestRunner.
